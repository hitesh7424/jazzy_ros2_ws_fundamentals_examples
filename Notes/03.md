# ROS 2 Subscriber and Launch File

This document provides instructions for creating a Python subscriber node and a bash launch file in ROS 2.

## Running the Nodes

### Run the Publisher Node

Run the `py_minimal_publisher` node from the `ros2_fundamental_examples` package:

```bash
ros2 run ros2_fundamental_examples py_minimal_publisher.py
```

### Run the Subscriber Node

Run the `py_minimal_subscriber` node from the `ros2_fundamental_examples` package:

```bash
ros2 run ros2_fundamental_examples py_minimal_subscriber.py
```

## Node Commands

### List all active nodes

```bash
ros2 node list
```

Example output:

```bash
- /minimal_py_publisher
- /minimal_py_subscriber
```

## Topics Covered

- **Create Python subscriber**: Learn how to create a Python subscriber node to receive messages.
- **Create bash launch file**: Understand how to create a bash launch file to run multiple nodes.
