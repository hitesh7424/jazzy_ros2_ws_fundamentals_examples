# Store ROS Workspace on GitHub

## Topics Covered

- How to store your ROS 2 workspace on GitHub.

### Steps

1. **Install Git**

   ```bash
   sudo apt install git -y
   ```

2. **Verify Git Installation**

   ```bash
   git --version
   ```

3. **Configure Git**

   ```bash
   git config --global user.name "Hitesh"
   git config --global user.email "hitesh164722@gmail.com"
   git config --global init.defaultBranch main
   ```

4. **Initialize Git Repository**

   ```bash
   git init
   ```

5. **Add Files to Staging Area**

   ```bash
   git add .
   ```

6. **Commit Changes**

   ```bash
   git commit -m "Initial commit"
   ```

7. **Fetch and Pull Changes**

   - Fetch updates from the remote repository:

     ```bash
     git fetch
     ```

   - Check the status of your repository:

     ```bash
     git status
     ```

   - Pull the latest changes:

     ```bash
     git pull
     ```

8. **Push Changes to GitHub**

   - Add the remote repository:

     ```bash
     git remote add origin <repository-url>
     ```

   - Push your changes:

     ```bash
     git push -u origin main
     ```

### Notes

- Ensure you have a GitHub account and a repository created to push your workspace.
- Use `git branch` to manage branches effectively.
- Regularly commit and push changes to keep your repository updated.
