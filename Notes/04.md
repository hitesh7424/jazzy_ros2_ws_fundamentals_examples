# Create Publisher in C++

## Topic Covered

- How to create a publisher in C++ for ROS 2.

### Key Points

- Learn the basics of creating a publisher node in C++.
- Understand the structure of a ROS 2 C++ package.
- Implement and test the publisher node.

### Steps to Create a Publisher Node

1. **Create a ROS 2 Package**
   - Use the `ros2 pkg create` command to create a new package.
   - Ensure the package dependencies include `rclcpp` and `std_msgs`.

2. **Write the Publisher Node**
   - Create a C++ file (e.g., `publisher_node.cpp`) in the `src` directory.
   - Include the necessary headers for `rclcpp` and message types.
   - Define a class that inherits from `rclcpp::Node`.
   - Create a publisher using `create_publisher` and publish messages in a timer callback.

3. **Build the Package**
   - Add the source file to `CMakeLists.txt`.
   - Build the package using `colcon build`.

4. **Run the Node**
   - Source the workspace and use `ros2 run` to execute the node.
