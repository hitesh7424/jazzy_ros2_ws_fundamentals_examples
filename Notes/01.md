# ROS Basics

This document provides a step-by-step guide to setting up and working with ROS 2.

## Workspace Setup

1. **Create a workspace directory:**

   ```bash
   mkdir -p ros2_ws/src
   cd ros2_ws/src
   ```

2. **Create a new ROS 2 package:**

   ```bash
   ros2 pkg create --build-type ament_cmake --license Apache-2.0 ros2_fundamental_examples
   ```

3. **Navigate back to the workspace root:**

   ```bash
   cd ..
   ```

4. **Build the workspace:**

   ```bash
   colcon build
   ```

5. **Source the setup file:**

   ```bash
   source ./install/setup.bash
   ```

6. **List available ROS 2 packages:**

   ```bash
   ros2 pkg list
   ```

7. **Build a specific package:**

   ```bash
   colcon build --packages-select ros2_fundamental_examples
   ```

## Installing Required Dependencies

Run the following command to install necessary ROS 2 dependencies:

```bash
sudo apt-get install -y \
  ros-${ROS_DISTRO}-ros-gz \
  ros-${ROS_DISTRO}-gz-ros2-control \
  ros-${ROS_DISTRO}-gz-ros2-control-demos \
  ros-${ROS_DISTRO}-joint-state-publisher-gui \
  ros-${ROS_DISTRO}-moveit \
  ros-${ROS_DISTRO}-xacro \
  ros-${ROS_DISTRO}-ros2-control \
  ros-${ROS_DISTRO}-ros2-controllers \
  libserial-dev \
  python3-pip
```

## Notes

- Replace `${ROS_DISTRO}` with your ROS 2 distribution (e.g., `humble`, `foxy`, etc.).
- Ensure that you have sourced the ROS 2 setup file before running any ROS 2 commands.
- Use `colcon build` to compile your workspace and `source ./install/setup.bash` to make the packages available in your environment.
