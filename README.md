# ROS 2 Fundamentals Examples

## Playlist Overview

**Playlist Link:** [Jazzy - Beginners - ROS 2 Fundamentals](https://youtube.com/playlist?list=PLNWNEEf8BvG64FVZT4IdieI1PuYnHkUrt)

**Description:**
This series is designed to equip you with the foundational ROS 2 skills essential for modern robotics development. By the end of this series, you’ll have a strong grasp of key ROS 2 concepts, empowering you to confidently create, run, and manage ROS 2 nodes, communicate between components, and build scalable robotics applications.

## Description

this package includes minimalist ROS 2  code to demonstrated important ROS 2  concepts and patterns

- Following ROS 2 Pythong style guidelines

## Prerequisites

- ROS 2 installed
- Python 3
- Created a ROS 2  workspace (`ros_ws`)
